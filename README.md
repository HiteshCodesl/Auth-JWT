A simple authentication application built with **Next.js**, **Tailwind CSS**, and **MongoDB**, featuring login, signup, and user profile management.

## Features

- User signup and login
- JWT-based authentication
- Protected client and API routes
- Change user name and password
- Responsive and modern UI

---

## Tech Stack

- **Frontend:** Next.js 13+, React, Tailwind CSS
- **Backend:** Next.js API routes
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT, localStorage

---

## Prerequisites

Make sure you have installed:

- Node.js
- npm
- MongoDB

---

### 1. Clone the repository

git clone https://github.com/HiteshCodesl/Auth-JWT.git  

### 2. Change Dirctory    
  
cd auth-jwt   

### 3. Install Dependencies 
  
npm install    

### env variables

### MongoDB URL
MONGODB_URL= 

### JWT secret key
JWT_SECRET=

### Backend URL for frontend requests
NEXT_PUBLIC_BACKEND_URL= 


### Run Server

npm run dev
